## [1.0.0] - 2026-03-27

### Added
- Browse and search the full Reactor catalog by name, author, and category
- One-click install, update, and uninstall of Reactor atoms
- Auto-detection of DaVinci Resolve Fusion installation path on macOS and Windows
- Custom repository support (GitLab, GitHub, local folders, HTTP servers)
- Repository category labels with sidebar filtering
- Import/export repository configurations as JSON
- Real-time installation progress with per-file tracking
- Retry logic for failed downloads (up to 5 attempts)
- Update detection for installed atoms
- Settings panel for manual Fusion path configuration
